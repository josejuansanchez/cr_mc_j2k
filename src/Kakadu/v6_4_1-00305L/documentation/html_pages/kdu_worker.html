kdu_worker

Java: class Kdu_worker

[Declared in "../coresys/common/kdu_threads.h"]

Synopsis

Pure abstract base class, which exists only to define the do_job function. Actual work which is to be supplied as jobs to the kdu_thread_entity::add_jobs function must be described by a class derived from this one.
