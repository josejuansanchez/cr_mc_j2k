video_show Software Manual
==========================

please send comments and additions to vruiz@ual.es

1. Compilation (only undex Unix)
--------------------------------
Makefile for GNU make is provided in the "video_show" directory. Just go to
that directory and type "make all". Note that the EXE variable (which
references the output executables directory) should be correctly defined.

2. Execution
------------
After compilation, try "video_show -help" to know the command line parameters
that can be supplied.
